InvestMania 使用說明

基本使用方法

1. 在輸入框中輸入欲查詢的股票代碼（例如：2330.TW）

2. 點擊「進行預測」按鈕

3. 系統將自動分析並顯示預測結果

預測指標說明

技術指標

成交量

• 當日成交量 > 最近20日均量 1.5 倍：可能是買盤積極信號

• 當日成交量 < 最近20日均量 0.5 倍：市場活躍度下降

MA (移動平均線)

• MA5：5日移動平均線

• MA20：20日移動平均線

• 短期均線上穿長期均線：黃金交叉，可能上漲

• 短期均線下穿長期均線：死亡交叉，可能下跌

RSI (相對強弱指標)

• RSI > 70：超買，可能回調

• RSI < 30：超賣，可能反彈

• 30 ≤ RSI ≤ 70：正常波動

MACD (移動平均趨勢指標)

• MACD > 0 且向上交叉信號線：買入信號

• MACD < 0 且向下交叉信號線：賣出信號

布林通道

• 價格接近上軌：可能超買，市場情緒可能過於樂觀，需注意回調風險

• 價格接近下軌：可能超賣，市場情緒可能過於悲觀，潛在反彈機會較高

• 價格在中軌附近：表示價格趨於穩定，是多空相對平衡的狀態

馬可夫鏈狀態說明

• 大漲(++)：單日漲幅 ≥ 6%

• 小漲(+)：0.5% ≤ 單日漲幅 < 6%

• 平盤(~)：-0.5% ≤ 單日漲跌幅 < 0.5%

• 小跌(-)：-6% ≤ 單日跌幅 < -0.5%

• 大跌(--)：單日跌幅 < -6%

預測結果解讀

• 預測價格：下一交易日收盤價預測

• 預期報酬率：價格變動百分比

• 預測區間：95% 信賴區間

• 狀態預測：馬可夫鏈趨勢

注意事項

• 本系統提供的預測僅供參考，不構成投資建議

• 投資決策應考慮個人風險承受能力和市場實際狀況

• 過去的表現不代表未來的走勢

• 建議配合其他分析工具和專業意見使用
